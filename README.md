# Movie search

A vanilla JavaScript movie search app built with the TMDB API.
No frameworks, no build tools — just HTML, CSS, and JS.

Live: https://yourusername.github.io/movie-search

## Features

- Search movies and TV shows via TMDB API
- Filter by type (film / TV)
- Filter by release year
- Sort by relevance, year, or title
- Skeleton loading state
- Poster fallback for missing images

## Concepts practised

- async / await and fetch with Bearer token auth
- DOM manipulation with querySelector and innerHTML
- Array methods — filter, map, sort
- ES6+ — destructuring, template literals, nullish coalescing
- Client-side filtering without re-fetching

## Setup

1. Get a free API token at themoviedb.org
2. Create env.js with: const API_TOKEN = 'your_token'
3. Open index.html in a browser

## Stack

Vanilla HTML + CSS + JavaScript — no dependencies
